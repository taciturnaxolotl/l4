# The L4 cache

This is my own image cdn built on cloudflare r2 mainly so I can have fast optimized images on my blog.

## Docs

Create `.env`:

```env
SLACK_BOT_TOKEN=token
SLACK_SIGNING_SECRET=secret
AUTH_TOKEN=token
```

```bash
bun install
bun start
```
